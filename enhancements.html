Enhancements

1. Navigation Bar Animations:

Description: Hovering on the navbar elements highlights them with a smooth transition (0.1s) and opens the associated dropdown menu containing anchors to other pages. The elements in the dropdown menu have a smooth highlighting effect as well. These effects are achieved using a combination of descendant combinators and psuedo-classes, which are beyond the scope of the curriculum.

Associated Code: .dropdown:hover .dropdown-content {display: block;} (style.css, line 87-125)

Associated Element: Navigation Bar

Source: W3Schools - CSS Transitions

2. Image Overlay Effects:

Description: Hovering on the images in the product showcase on the homepage blurs them with a smooth transition (0.3s) and displays a text overlay containing the link to the associated product page. These effects are achieved using psuedo-classes and properties such as transition and opacity, which are beyond the scope of the curriculum.

Associated Code: .products-item:hover img {opacity: 0.5;} (style.css line 167-208)

Associated Element: Products Showcase

Source: W3Schools - Image Hover Overlay

3. Grid Layout:

Description: The product glossaries in the product pages are arranged in a grid-like pattern using the grid property on the container. Additionally, the images have a smooth enlargement effect upon hovering thanks to the use of psuedo classes and properties such as transition and scale. These concepts are beyond the scope of the curriculum.

Associated Code: .product-gallery {display: grid;} (style.css line 538, 653-666)

Associated Element: Products Gallery

Source: W3Schools - CSS Grid Layout Module

4. Animated Gradient Buttons:

Description: The buttons in the aside sections of workshop and promo pages have moving gradient backgrounds and increase in opacity and size when hovered over. These features are implemented using @keyframes rules, which are beyond the scope of the curriculum.

Associated Code: @keyframes gradientMove { 0% { background-position: 0% 50%; } 50% { background-position: 100% 50%; } 100% { background-position: 0% 50%; } }

Associated Element: Workshops Aside

Source: W3Schools - CSS @keyframes Rules
